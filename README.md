# APP A2T



## Mô Tả
Cài đặt Games Homebrew đơn giản.

Có Code Devices để xác thực máy user.

Sử dụng bộ Database riêng và tự động cập nhập theo App

Game được cập nhập hằng ngày!

## NX-META Database
Bạn cũng có thể tải DB [ở đây](https://github.com/tuannahh/A2T_DB)

Sau khi tải xong bạn chép vào thẻ nhớ 

```bash
sdmc:/switch/A2T/offline_db/
```
